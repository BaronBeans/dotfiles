## my fave shortcuts:

### vim
* gt and gT to :tabn and :tabp
* :tab ball to open all current buffers in their own tabs
* <C-s> split vertical
* <C-\> split horizontal

### tmux
* <prefix> c - create new window
* <prefix> n - next window
* <prefix> p - previous window
* <prefix> " - new split horizontal
* <prefix> % - new split horizontal
